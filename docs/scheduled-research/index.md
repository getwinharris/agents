# Scheduled Research

This directory contains source-grounded internal research maintained by approved recurring workflows.

Research records use ordinary Markdown without YAML frontmatter. Add categories only when supported by verified repository evidence and current official primary sources.

## Workflow

1. Verify the current repository implementation and applicable `AGENTS.md` instructions.
2. Cross-check findings against current official primary sources.
3. Search existing research, issues, and pull requests before adding or updating a record.
4. Link confirmed actionable gaps to focused GitHub issues and implementation pull requests.
5. Update this index only when a durable research category or status changes.
